# Red Teaming Projects

This repository contains a structured collection of hands-on cybersecurity projects, built as part of a progressive learning path.

## Purpose

The goal of this repository is to develop practical security and networking skills by building tools from scratch. Each project focuses on understanding how systems communicate, how services are discovered, and how real-world tools operate under the hood.

## Structure

Projects are organized by difficulty and topic:

- **Level 1 – Basics**: Foundational networking tools and simple utilities  
- **Level 2 – Scanning & Enumeration**: Port scanners, service detection, and reconnaissance tools  
- *(Future levels will expand into more advanced topics)*

Each project is contained in its own folder and includes:
- Source code
- A brief explanation of the concept
- Usage instructions (when applicable)

## Notes

This repository is intended for educational purposes only. All tools should be used responsibly and only in authorized environments.
